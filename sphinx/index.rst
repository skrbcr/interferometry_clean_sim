Interferometry CLEAN Simulation documentation
=============================================

**Interferometry CLEAN Simulation** is a package for simulating the CLEAN algorithm for radio interferometry imaging.

The repository is available on `GitHub <https://github.com/skrbcr/interferometry_clean_sim>`_.
